suma :: (Int, Int) -> Int Source

(suma (x,y)) es la suma e x e y. Por ejemplo,

(suma (2,3)) == 5

deCeroA :: Int -> [Int] Source

(deCeroA n) es la lista de los números desde 0 hasta n. Por ejemplo, 

deCeroA 5 == [0,1,2,3,4,5]

suma' :: Int -> Int -> Int Source

(suma' x y) es la suma de x e y. Por ejemplo,

suma' 2 3   == 5
(suma' 2) 3 == 5

mult :: Int -> Int -> Int -> Int Source

(mult x y z) es el producto de x, y y z. Por ejemplo,

mult 2 5 7     == 70
(mult 2) 5 7   == 70
((mult 2) 5) 7 == 70

suc :: Int -> Int Source

(suc n) es el siguiente de n. Por ejemplo,

suc 5 == 6
